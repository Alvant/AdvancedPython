## –ó–∞–¥–∞—á–∞ 3 (¬´–í—ã—á–µ—Ç—ã –ø–æ –º–æ–¥—É–ª—é n¬ª‚Äé)

–ú–æ–¥—É–ª—å–Ω–∞—è –∞—Ä–∏—Ñ–º–µ—Ç–∏–∫–∞ ‚Äî –æ–ø–µ—Ä–∞—Ü–∏–∏ —Å –æ—Å—Ç–∞—Ç–∫–∞–º–∏ –ø–æ –Ω–µ–∫–æ—Ç–æ—Ä–æ–º—É –º–æ–¥—É–ª—é.
–ù–∞–ø—Ä–∏–º–µ—Ä, *–ø–æ –º–æ–¥—É–ª—é `3`* –ø—è—Ç—ë—Ä–∫–∞ ‚Äî —ç—Ç–æ —Ç–æ –∂–µ —Å–∞–º–æ–µ, —á—Ç–æ –∏ –¥–≤–æ–π–∫–∞.
–ü–æ—Ç–æ–º—É —á—Ç–æ —É –Ω–∏—Ö —Å–æ–≤–ø–∞–¥–∞—é—Ç –æ—Å—Ç–∞—Ç–∫–∏ –æ—Ç –¥–µ–ª–µ–Ω–∏—è –Ω–∞ `3` (–≤ –æ–±–æ–∏—Ö —Å–ª—É—á–∞—è—Ö –æ—Å—Ç–∞—Ç–æ–∫ —Ä–∞–≤–µ–Ω `2`, —Ç–∞–∫ –∫–∞–∫ `5 - 2 = 3` ‚Äî –∫—Ä–∞—Ç–Ω–æ `3`, –∏ `2 - 2 = 0` ‚Äî —Ç–æ–∂–µ –∫—Ä–∞—Ç–Ω–æ `3`).
–í—Å–µ–≥–æ, –æ—á–µ–≤–∏–¥–Ω–æ, –µ—Å—Ç—å `n - 1` —Ä–∞–∑–ª–∏—á–Ω—ã—Ö –æ—Å—Ç–∞—Ç–∫–æ–≤ –ø–æ –º–æ–¥—É–ª—é`n`.
–ù–∞–ø—Ä–∏–º–µ—Ä, –æ—Å—Ç–∞—Ç–∫–∏ –ø–æ –º–æ–¥—É–ª—é `3` ‚Äî —ç—Ç–æ `0`, `1` –∏ `2`.

–ù–∞–¥–æ —Ä–µ–∞–ª–∏–∑–æ–≤–∞—Ç—å –∫–ª–∞—Å—Å `Residue` –æ—Å—Ç–∞—Ç–∫–∞ –ø–æ –º–æ–¥—É–ª—é `n`.
–î–æ–ª–∂–Ω–∞ –±—ã—Ç—å –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å —Å—Ä–∞–≤–Ω–µ–Ω–∏—è –Ω–∞ —Ä–∞–≤–µ–Ω—Å—Ç–≤–æ –æ—Å—Ç–∞—Ç–∫–æ–≤ *–ø–æ –æ–¥–Ω–æ–º—É –∏ —Ç–æ–º—É –∂–µ –º–æ–¥—É–ª—é*.
–¢–∞–∫–∂–µ –¥–æ–ª–∂–Ω—ã –±—ã—Ç—å —Ä–µ–∞–ª–∏–∑–æ–≤–∞–Ω—ã —Å–ª–µ–¥—É—é—â–∏–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ —Å –æ–±—ä–µ–∫—Ç–∞–º–∏ –æ—Å—Ç–∞—Ç–∫–æ–≤ –ø–æ –æ–¥–Ω–æ–º—É –∏ —Ç–æ–º—É –∂–µ –º–æ–¥—É–ª—é:
* —Å–ª–æ–∂–µ–Ω–∏–µ
* –≤—ã—á–∏—Ç–∞–Ω–∏–µ
* —É–º–Ω–æ–∂–µ–Ω–∏–µ

–ü—Ä–∏ –ø–æ–ø—ã—Ç–∫–µ –≤—ã–ø–æ–ª–Ω–∏—Ç—å –ª—é–±—É—é –∏–∑ –æ–ø–∏—Å–∞–Ω–Ω—ã—Ö –æ–ø–µ—Ä–∞—Ü–∏–π —Å –æ—Å—Ç–∞—Ç–∫–∞–º–∏ –ø–æ —Ä–∞–∑–Ω—ã–º –º–æ–¥—É–ª—è–º –¥–æ–ª–∂–Ω–∞ –≤—ã–±—Ä–∞—Å—ã–≤–∞—Ç—å—Å—è –æ—à–∏–±–∫–∞ `TypeError`.

–ï—â—ë –Ω–∞–¥–æ —Å–¥–µ–ª–∞—Ç—å —Ç–∞–∫, —á—Ç–æ–±—ã –±—ã–ª–æ –ø–æ–Ω—è—Ç–Ω–æ–µ –∏ "—á–∏—Ç–∞–µ–º–æ–µ" –æ–ø–∏—Å–∞–Ω–∏–µ –æ–±—ä–µ–∫—Ç–∞ –æ—Å—Ç–∞—Ç–∫–∞ –ø—Ä–∏ –≤—ã–≤–æ–¥–µ –Ω–∞ —ç–∫—Ä–∞–Ω (—á–µ—Ä–µ–∑ —Ñ—É–Ω–∫—Ü–∏—é `print` –∏–ª–∏ –ø—Ä–æ—Å—Ç–æ –ø—Ä–∏ –≤—ã–∑–æ–≤–µ –≤ —è—á–µ–π–∫–µ –Æ–ø–∏—Ç–µ—Ä-–Ω–æ—É—Ç–±—É–∫–∞).

–í–æ–∑–º–æ–∂–Ω—ã–π –ø—Ä–∏–º–µ—Ä —Ä–∞–±–æ—Ç—ã:

## –ó–∞–¥–∞—á–∞ 4 (¬´‚ÄéCustomDict¬ª‚Äé)

## –ó–∞–¥–∞—á–∞ 7 (¬´Chemistry¬ª‚Äé)

## –ó–∞–¥–∞—á–∞ 9* (¬´‚ÄéSecret Storage¬ª‚Äé)

–ù–∞–ø–∏—à–∏—Ç–µ –∫–ª–∞—Å—Å `Storage`, –∫–æ—Ç–æ—Ä—ã–π –ø—Ä–∏ —Å–æ–∑–¥–∞–Ω–∏–∏ –ø—Ä–∏–Ω–∏–º–∞–µ—Ç –Ω–∞ –≤—Ö–æ–¥ –∫–∞–∫—É—é-—Ç–æ –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—é –≤ –≤–∏–¥–µ —Å–ª–æ–≤–∞—Ä—è.
–ü—Ä–∏ —ç—Ç–æ–º —É –æ–±—ä–µ–∫—Ç–æ–≤ –∫–ª–∞—Å—Å–∞ `Storage` –¥–æ–ª–∂–Ω—ã –±—ã—Ç—å –ø—Ä–∏–≤–∞—Ç–Ω—ã–µ –∞—Ç—Ä–∏–±—É—Ç—ã `_reserve_data` –∏ `_secret_data` ‚Äî —Ç–æ–∂–µ —Å–ª–æ–≤–∞—Ä–∏, —Ñ–∏–∫—Å–∏—Ä–æ–≤–∞–Ω–Ω—ã–µ.
–ù–∞–ø—Ä–∏–º–µ—Ä, –∫–æ–Ω—Å—Ç—Ä—É–∫—Ç–æ—Ä –∫–ª–∞—Å—Å–∞ `Storage` –º–æ–∂–µ—Ç –≤—ã–≥–ª—è–¥–µ—Ç—å —Ç–∞–∫:
```python

    def __init__(self, data: Dict[str, Any]):
```

–ù–∞–¥–æ —Å–¥–µ–ª–∞—Ç—å —Ç–∞–∫, —á—Ç–æ–±—ã –º–æ–∂–Ω–æ –±—ã–ª–æ –æ–±—Ä–∞—â–∞—Ç—å—Å—è –ø–æ –∫–ª—é—á—É –∫ –æ–±—ä–µ–∫—Ç—É –∫–ª–∞—Å—Å–∞ `Storage`. –ü—Ä–∏ —ç—Ç–æ–º

* –µ—Å–ª–∏ –∫–ª—é—á–∞ –Ω–µ—Ç –≤ `_data`, —Ç–æ –¥–æ–ª–∂–µ–Ω –ø—Ä–æ–∏–∑–æ–π—Ç–∏ –ø–æ–∏—Å–∫ –∫–ª—é—á–∞ –≤ `_reserve_data`, –∏, –µ—Å–ª–∏ –∫–ª—é—á –µ—Å—Ç—å —Ç–∞–º, –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç—Å—è —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—â–µ–µ –∑–Ω–∞—á–µ–Ω–∏–µ –∏–∑ `_reserve_data`
* –µ—Å–ª–∏ –∂–µ –∫–ª—é—á–∞ –Ω–µ—Ç –Ω–∏ –≤ `_data`, –Ω–∏ –≤ `_reserve_data`, —Ç–æ –¥–æ–ª–∂–Ω–∞ –≤—ã–±—Ä–∞—Å—ã–≤–∞—Ç—å—Å—è –æ—à–∏–±–∫–∞ `KeyError`

–ï—â—ë –Ω–∞–¥–æ —Å–¥–µ–ª–∞—Ç—å —Ç–∞–∫, —á—Ç–æ–±—ã –±—ã–ª –∑–∞–∫—Ä—ã—Ç –¥–æ—Å—Ç—É–ø –∏–∑–≤–Ω–µ –∫ `_secret_data` ‚Äî –Ω–∞ —Ç–æ –æ–Ω–∞ –∏ "secret" :)

```python
storage = Storage({'apple': 2})
```

## –ó–∞–¥–∞—á–∞ 10* (¬´–ö–≤–∞–Ω—Ç–æ–≤–∞–Ω–Ω—ã–π —Å–ø–∏—Å–æ–∫¬ª‚Äé)

–û *–∫–≤–∞–Ω—Ç–∏–∑–∞—Ü–∏–∏* –º–æ–∂–Ω–æ –¥—É–º–∞—Ç—å –∫–∞–∫ –æ –ø–µ—Ä–µ–≤–æ–¥–µ —á–∏—Å–ª–∞ –∏–∑ —Ç–∏–ø–∞ `float32` –≤ `int` (int16/int8/int4/int2).
–¢–æ –µ—Å—Ç—å –∏–∑ "–Ω–µ–ø—Ä–µ—Ä—ã–≤–Ω–æ–≥–æ" —Ç–∏–ø–∞ –≤ –¥–∏—Å–∫—Ä–µ—Ç–Ω—ã–π.

–ö–≤–∞–Ω—Ç–∏–∑–∞—Ü–∏—è –º–æ–∂–µ—Ç –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å—Å—è –¥–ª—è —É–º–µ–Ω—å—à–µ–Ω–∏—è —Ä–∞–∑–º–µ—Ä–∞ –¥–∞–Ω–Ω—ã—Ö.
–ï—â—ë –æ–¥–Ω–æ –∏–∑ –≤–æ–∑–º–æ–∂–Ω—ã—Ö –ø—Ä–∏–º–µ–Ω–µ–Ω–∏–π –∫–≤–∞–Ω—Ç–∏–∑–∞—Ü–∏–∏ ‚Äî —É—Å–∫–æ—Ä–µ–Ω–∏–µ –≤—ã—á–∏—Å–ª–µ–Ω–∏–π: –Ω–µ–∫–æ—Ç–æ—Ä—ã–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ —Å –∫–≤–∞–Ω—Ç–æ–≤–∞–Ω–Ω—ã–º–∏ –º–∞—Ç—Ä–∏—Ü–∞–º–∏ –º–æ–≥—É—Ç –∏—Å–ø–æ–ª–Ω—è—Ç—å—Å—è –±—ã—Å—Ç—Ä–µ–µ, —á–µ–º —Å –º–∞—Ç—Ä–∏—Ü–∞–º–∏ –≤ –ø–æ–ª–Ω–æ–π —Ç–æ—á–Ω–æ—Å—Ç–∏ float32 (–ø—Ä–∞–≤–¥–∞, –Ω–∞ –ø—Ä–∞–∫—Ç–∏–∫–µ –µ—â—ë –≤–∞–∂–Ω–æ, —á—Ç–æ–±—ã –ø—Ä–æ—Ü–µ—Å—Å–æ—Ä –ø–æ–¥–¥–µ—Ä–∂–∏–≤–∞–ª —ç—Ç–∏ —Å–∞–º—ã–µ –±–æ–ª–µ–µ –æ–ø—Ç–∏–º–∞–ª—å–Ω—ã–µ –≤–∞—Ä–∏–∞–Ω—Ç—ã –æ–ø–µ—Ä–∞—Ü–∏–π).

–°—É—â–µ—Å—Ç–≤—É–µ—Ç –Ω–µ—Å–∫–æ–ª—å–∫–æ —Å–ø–æ—Å–æ–±–æ–≤, "—Å—Ç—Ä–∞—Ç–µ–≥–∏–π" –∫–≤–∞–Ω—Ç–∏–∑–∞—Ü–∏–∏.
–†–∞—Å—Å–º–æ—Ç—Ä–∏–º —Å–ª–µ–¥—É—é—â–∏–π ("–±–µ–∑ –Ω–∞–≤–æ—Ä–æ—Ç–æ–≤").
–ü—É—Å—Ç—å –µ—Å—Ç—å —Å–ø–∏—Å–æ–∫ `values` —á–∏—Å–µ–ª –≤ —Ç–æ—á–Ω–æ—Å—Ç–∏ `float` –∏ —Ñ–ª–æ—Ç–æ–≤—ã–π –∂–µ –º–Ω–æ–∂–∏—Ç–µ–ª—å `alpha`.
–¢–æ–≥–¥–∞ –¥–ª—è —Ç–æ–≥–æ, —á—Ç–æ–±—ã –ø–æ—Å—Ç—Ä–æ–∏—Ç—å *–∫–≤–∞–Ω—Ç–æ–≤–∞–Ω–Ω—ã–π —Å–ø–∏—Å–æ–∫*, –Ω–∞–¥–æ –¥–ª—è –∫–∞–∂–¥–æ–≥–æ —á–∏—Å–ª–∞ `x` –∏–∑ `values` —Å–¥–µ–ª–∞—Ç—å —Å–ª–µ–¥—É—é—â–µ–µ (–ø—Å–µ–≤–¥–æ–∫–æ–¥ –ø—Ä–µ–¥–ª–∞–≥–∞–µ–º–æ–≥–æ –∞–ª–≥–æ—Ä–∏—Ç–º–∞ –∫–≤–∞–Ω—Ç–∏–∑–∞—Ü–∏–∏): 
```python
# –í—Ö–æ–¥: x (float), alpha, num_bits
# –í—ã—Ö–æ–¥: x_quant (int)

# –î–µ–ª–∏–º –Ω–∞ alpha, —á—Ç–æ–±—ã –ø—Ä–∏–π—Ç–∏ –∫ –Ω–µ–∫–æ–µ–º—É –∫–æ–º–ø—Ä–æ–º–∏—Å—Å—É –º–µ–∂–¥—É
# –ø–æ—Ç–µ—Ä—è–º–∏ –ø—Ä–∏ –∫–≤–∞–Ω—Ç–∏–∑–∞—Ü–∏–∏, —Ç–æ –µ—Å—Ç—å —á–∏—Å–ª–æ–º "–∫–ª—ç–º–ø–Ω—É—Ç—ã—Ö" –∑–Ω–∞—á–µ–Ω–∏–π
#     (–º–∞–ª—ã–µ –ø–æ—Ç–µ—Ä–∏ –ø—Ä–∏ –±–æ–ª—å—à–∏—Ö alpha ‚Äî —Ö–æ—Ä–æ—à–æ)
# –∏ —Ç–æ—á–Ω–æ—Å—Ç—å—é –∫–≤–∞–Ω—Ç–∏–∑–∞—Ü–∏–∏, —Ç–æ –µ—Å—Ç—å —Ä–∞–∑–Ω–∏—Ü–µ–π –º–µ–∂–¥—É –æ—Ä–∏–≥–∏–Ω–∞–ª—å–Ω—ã–º float –∏ –∫–≤–∞–Ω—Ç–æ–≤–∞–Ω–Ω—ã–º int —á–∏—Å–ª–æ–º
#     (–º–∞–ª–∞—è —Ç–æ—á–Ω–æ—Å—Ç—å –ø—Ä–∏ –±–æ–ª—å—à–∏—Ö alpha ‚Äî –ø–ª–æ—Ö–æ)
x_scaled = x / alpha

# –°–æ–±—Å—Ç–≤–µ–Ω–Ω–æ "–∫–ª—ç–º–ø" ("–æ–±—Ä–µ–∑–∞–Ω–∏–µ" —Ç–µ—Ö –∑–Ω–∞—á–µ–Ω–∏–π, —á—Ç–æ –∑–∞ –ø—Ä–µ–¥–µ–ª–∞–º–∏),
# —á—Ç–æ–±—ã –ø–æ—Ç–æ–º —Ñ–∏–∫—Å–∏—Ä–æ–≤–∞–Ω–Ω—ã–π –æ—Ç—Ä–µ–∑–æ–∫ [-1, 1] –ø–µ—Ä–µ–≤–µ—Å—Ç–∏ –≤ –æ—Ç—Ä–µ–∑–æ–∫ [min_int, max_int]
x_clamped = x_scaled.clamp(min=-1, max=1)

# –°–æ—Ö—Ä–∞–Ω—è–µ–º –∑–Ω–∞–∫ —á–∏—Å–ª–∞ (–Ω–∞ —ç—Ç–æ –ø–æ–Ω–∞–¥–æ–±–∏—Ç—Å—è –æ–¥–∏–Ω –±–∏—Ç)
sign = x_clamped.sign()
x_abs = x_clamped.abs()

# –ö–≤–∞–Ω—Ç–æ–≤–∞–Ω–∏–µ
x_quant = x_abs.mul(2 ** num_bits - 1)
x_quant = x_quant.round()
```

–ß—Ç–æ–±—ã –ø–æ—Ç–æ–º –ø–æ–ª—É—á–∏—Ç—å –∏–∑ –∫–≤–∞–Ω—Ç–æ–≤–∞–Ω–Ω–æ–≥–æ `x` —Ñ–ª–æ—Ç–æ–≤–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ, –Ω–∞–¥–æ –ø—Ä–æ–≤–µ—Å—Ç–∏ "–¥–µ–∫–≤–∞–Ω—Ç–∏–∑–∞—Ü–∏—é":
```python
# –ü–µ—Ä–µ–≤–æ–¥–∏–º –≤ [-1, 1]
x = x_quant.div(2 ** num_bits - 1)

# –í–æ—Å—Å—Ç–∞–Ω–∞–≤–ª–∏–≤–∞–µ–º –∑–Ω–∞–∫
x = sign * x

# –ü–µ—Ä–µ–≤–æ–¥–∏–º –≤ –Ω—É–∂–Ω—ã–µ –ø—Ä–µ–¥–µ–ª—ã
x = x * alpha
```

–ò—Ç–∞–∫, –≤ –∑–∞–¥–∞–Ω–∏–∏ —Ç—Ä–µ–±—É–µ—Ç—Å—è —Ä–µ–∞–ª–∏–∑–æ–≤–∞—Ç—å –∫–ª–∞—Å—Å –∫–≤–∞–Ω—Ç–æ–≤–∞–Ω–Ω–æ–≥–æ —Å–ø–∏—Å–∫–∞ –∑–Ω–∞—á–µ–Ω–∏–π `QuantVector`.
–î–æ–ª–∂–Ω–∞ –±—ã—Ç—å –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å —Å–ª–æ–∂–µ–Ω–∏—è –¥–≤—É—Ö —Å–ø–∏—Å–∫–æ–≤ —Å –ø–æ–º–æ—â—å—é –æ–ø–µ—Ä–∞—Ç–æ—Ä–∞ `+`.
